# 生徒カルテ

**未来の自分を助ける、担任の手元カルテ。**

生徒の名簿・指導や支援の記録・テスト成績・生徒同士の配慮関係を、お手元の **Excelファイル1冊** にまとめて管理する、中学校教員向けのツールです。病院の電子カルテのように、生徒の名前で呼び出して、その子のすべての情報を1画面で見られます。

- ✅ **登録不要・インストール不要** — フォルダを開くだけで動きます
- ✅ **データは常にExcelが正本** — ツールはそれを読み書きするだけ。Excelでの直接編集もOK
- ✅ **3年間つながる** — 生徒に不変のIDが付き、進級・クラス替えをしても記録が途切れません
- ✅ **インターネット送信ゼロ** — 入力した情報はPCの外に一切出ません（このリポジトリのコードで検証できます）

## 使い方（3ステップ)

1. **ダウンロード** — [Releases](../../releases) から最新のZIPをダウンロードして展開します
2. **開く** — フォルダ内の `index.html` を Chrome または Edge でダブルクリックして開きます
3. **はじめる** — 「テンプレートを発行」→ Excelで名簿を入力 → 「名簿ファイルを開く」

まず試したい場合は、`sample/生徒カルテ_サンプル.xlsx`（架空データ49名分）を開けば全機能を体験できます。

詳しい使い方はフォルダ内の `help.html`（使い方説明ページ）をご覧ください。

## 主な機能

- 🔍 **受付画面** — 氏名・ふりがな・IDで生徒を検索し、ワンクリックでカルテへ
- 📋 **生徒カルテ** — 基本情報／記録（指導・支援・面談…）／成績／関係の4タブ
- 🔀 **進級取り込み** — 新クラス編成表を貼り付けて一括更新。「同クラス不可」ペアの自動チェックつき
- 💯 **成績取り込み** — 採点用テンプレート発行 → 点数入力 → 貼り付けでID照合・一括登録
- 🖨 **カルテ印刷** — 三者懇談や担任引き継ぎの資料に（個人情報は既定で除外）
- 📊 **集計** — クラス別・クラブ別・誕生月・支援計画の人数

## 動作環境

| 環境 | 対応 |
|---|---|
| Chrome / Edge | フル機能（編集のたびに自動でExcelへ上書き保存） |
| Safari / Firefox | ダウンロード保存方式 |
| オフライン | 完全動作（外部リソースなし） |

## 個人情報の取り扱いについて

- 本ツールはデータを一切送信・収集しません。保存先はお使いのPC内のExcelファイルのみです
- 生徒情報を含むファイルの保管・持ち出しは、**所属校・自治体の情報管理規程に従ってください**
- 校務で使用する前に、私物ツールでの生徒情報の取り扱いが認められているか、所属校の規程をご確認ください
